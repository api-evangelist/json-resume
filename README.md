# Json ResumeThis is a repo for managing machine readable artifacts for the [Json Resume APIs](http://jsonresume.org/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Json Resume APIs](http://jsonresume.org/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://jsonresume.org/)- [Developer](http://jsonresume.org/)- [Twitter](https://twitter.com/jsonresume)- [GitHub](https://github.com/jsonresume)The [APIs.json](https://github.com/api-evangelist/json-resume/blob/master/apis.json) index provides links to all the relevant links for the Json Resume APIs, including the OpenAPI and Postman if they are present.
